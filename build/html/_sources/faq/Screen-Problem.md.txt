## display problem

### No signal source detected, no signal prompt:
- Is the HDMI cable connected securely?
- Does the Raspberry Pi start normally? When it starts normally, the red light is always on, and the green light flashes quickly
### black border appears on the screen:
1. Click on Raspberry in the upper right corner of the screen ==> Preferences ==> Raspberry Pi Configuration
2. Underscan select Enabied or Disabled
